# Housing.com Social Hook Examples — APPROVED

# These are real-world-tested examples that the social creative agent uses as few-shot prompts.
# Principle: Take a viral event → find the real estate angle INSIDE it → punchy Hinglish
#
# Format per example:
#   EVENT: [what happened]
#   CARD:  short punchy text that goes on the image card (≤ 3 lines)
#   CAPTION: post body text (Hinglish, ≤ 150 chars, ends with CTA or city link)
#   CITY_HINT: city name if a Housing.com city SRP link should be embedded
#
# Key rule: Do NOT copy-paste the event verbatim. Find the real estate metaphor or CTA
# that lives INSIDE the event. "Ghar" should feel like a natural punchline, not a bolted-on ad.

## SENSITIVITY RULES (agent must follow)
- Operation Sindoor: national pride angle ONLY. Never reference casualties, attack, or Pakistan directly.

- Raghav Chadha: tagging @raghavchadha is intentional (engagement bait) but keep it light/fun.
- Kejriwal/Judge: never name the judge. Generic/light framing only.
- Gau rakshak: SKIP entirely — too religiously charged.
- Godi media: very light touch — don't position as anti-BJP or pro-opposition.
