# Housing.com AI Content — Media Production Plan

> Creative Director Brief: 18 years across Cartoon Network, JWT India, Marvel Studios, Comedy Central.
> Mission: Turn real estate content from "informative" to "unmissable."

---

## Creative Philosophy

Real estate is boring. We make it feel like the most important decision of someone's life — because it is.

Rules I've lived by:
1. **Hook in 0–1.8 seconds or you're dead.** Not 3 seconds. 1.8.
2. **Emotion before information.** Make them feel the struggle/dream before you give data.
3. **Gen Z doesn't watch ads — they watch stuff that happens to be ads.**
4. **Trending audio + relatable visual + one sharp insight = viral.**
5. **Memes are the new press release.**

## Format Playbook

### 1. YouTube Shorts (60s max)

**Format**: Talking head (AI avatar or real) + B-roll cutaways + captions  
**Tool**: InVideo AI → Kling 3.0 for cinematic shots  
**Voice**: ElevenLabs (Priya voice — warm, confident, 26-year-old energy)  
**Structure**:
```
0:00–0:03  HOOK — punchy visual or bold text slam
0:03–0:15  SETUP — the problem/situation
0:15–0:45  CONTENT — 3 punchy points, max
0:45–0:55  TWIST or PUNCHLINE
0:55–1:00  CTA — "Follow for more" / "Link in bio"
```
**Trending sound strategy**: Always search trending audio on YouTube Shorts weekly. Overlay our content on top.

---

### 2. Instagram Reels (15–30s sweet spot)

**Tier A — Quick-fire facts** (InVideo AI, $1.20/reel):
- Hook text on screen, trending audio, 5–7 rapid cuts
- "5 things your builder WON'T tell you about the floor plan"

**Tier B — Cinematic property showcase** (Veo 3.1 or Kling 3.0, $7–9/reel):
- AI-generated cinematic walk-throughs
- "Your dream 2BHK in Baner, Pune — now under ₹75L"
- Slow dolly shots, golden hour lighting, luxury feel

**Tier C — Trending audio + relatable text** (Canva Pro + ElevenLabs, ~$0.50):
- Pop a trending audio, slap relatable text captions
- e.g., Diljit's "Goat" playing while "Me after finding a flat within budget finally"

---

### 3. Instagram Stories (24h format)

**Use case**: Quick updates, polls, FOMO triggers  
**Tools**: Canva Pro, Supermeme.ai for meme overlays  
**Templates**:
- Poll story: "Would you buy in Noida or Gurugram for ₹1Cr?" → drives DMs
- Countdown story: "RERA deadline alert — 3 days left for X project"
- Behind the brief: "Our AI found this trend at 2AM 👀" (transparency builds trust)
- Swipe-up (or link sticker): direct to blog/listing

**Cost**: ~$0.50 per story set (5 slides)

---

### 4. Instagram Carousels (2–10 slides)

**Best performing format for saves/shares on Instagram**  
**Tools**: Canva Pro + our existing pipeline carousel script  
**Structure**:
```
Slide 1: Thumb-stopping headline (bold, 6 words max)
Slide 2–8: One insight per slide, visual-first
Slide 9: Summary / "save this for later"
Slide 10: CTA + branding
```
**Topics that kill it**:
- "10 red flags in a builder agreement"
- "Bandra vs Andheri — where ₹2Cr gets you more"
- "RERA explained: before you sign anything, read this"

**Cost**: Effectively free (Canva) + script from existing pipeline

## Pipeline Integration: `media_producer` Agent

### `media_brief` JSON Structure

The existing pipeline will pass this JSON to the new `media_producer.py` agent:

```json
{
  "post_id": "uuid",
  "platform": "instagram_reel",
  "content_type": "video_reel | story | carousel | meme_video | meme_image | youtube_short",
  "script": "...",
  "hook_line": "POV: You've toured 47 flats...",
  "trending_audio_hint": "Diljit - Goat",
  "visual_style": "cinematic | meme | talking_head | text_on_screen",
  "emotion_target": "humor | aspiration | urgency | empowerment",
  "duration_seconds": 30,
  "meme_template": "distracted_boyfriend",
  "text_overlays": ["Line 1", "Line 2"],
  "cta": "Follow for daily real estate intel"
}
```

### `media_producer.py` Architecture

```
media_producer.py
├── route_by_content_type(brief)
│   ├── video_reel / youtube_short → VideoProducer
│   │   ├── SHORT: InVideo AI API (fast, $1.20)
│   │   └── CINEMATIC: Veo 3.1 or Kling 3.0 ($7–9)
│   ├── meme_video → MemeVideoProducer
│   │   └── Viggle AI (animate character) + ElevenLabs audio
│   ├── meme_image → MemeImageProducer
│   │   ├── Supermeme.ai API (template memes)
│   │   └── Imgflip API (classic templates, free)
│   ├── carousel → CarouselProducer
│   │   └── Canva API or HTML→PNG pipeline
│   └── story → StoryProducer
│       └── Canva API + ElevenLabs voiceover
└── save_media_to_storage(post_id, media_url)
    └── updates PublishedPostRecord.media_urls
```
